### Step 1: Check Agent Health Status

**What It Does:**
- Gateway connectivity check
- Agent availability verification
- Model status check
- Workspace health scan

**Output:** Agent health dashboard

**Agent Routing:** Main Agent (orchestrator)

### Step 2: Review Memory for Important Information
**Command:** Search MEMORY.md and workspace for:
- Meetings scheduled today
- Deadlines (today or upcoming)
- Recurring tasks
- Special events (birthdays, milestones)
- Important dates (project milestones, reviews)

**Search Queries:**
```
Search MEMORY.md for:
- "meetings today"
- "deadline"
- "February 18"
- "milestone"
- "recurring task"
- "special event"
```

**Agent Routing:** Main Agent (orchestrator)

---

### Step 3: Check Yesterday's Performance
**From:** PERFORMANCE_METRICS.md
- Check yesterday's metrics
- Tasks completed
- Issues encountered
- Improvements made

**Agent Routing:** Main Agent (orchestrator)

---

### Step 4: Review Cost Metrics (Last 7 Days)
**From:** PERFORMANCE_METRICS.md
- Cost summary
- Free model usage percentage
- Token efficiency
- Cost per task

**Agent Routing:** Main Agent (orchestrator)

---

### Step 5: Identify Today's Priorities
**From:** USERS_PREFERENCES.md & current pending tasks
- List tasks from memory
- Prioritize by importance/urgency
- Suggest agent assignments:
  - Atlas: Architecture planning
  - Nova: General tasks
  - Coder: Code-related
  - Orion: Debugging
  - Zen: Deep analysis
  - Flash: Quick tasks
  - Spark: Rapid response
  - Vision: Vision tasks

**Agent Routing:** Main Agent (orchestrator)

---

### Step 6: Identify Proactive Opportunities
**From:** HEARTBEAT.md and OPTIMIZATION_QUICK_START.md
- Skills not yet tested
- Workflows to document
- Optimization opportunities
- System improvements

**Agent Routing:** Main Agent (orchestrator)
