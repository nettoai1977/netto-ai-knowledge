## Twitter/X Integration

### How It Works
- OpenClaw uses browser automation (not API)
- Manual login in dedicated Chrome profile
- Uses host browser to avoid bot detection

### Capabilities
- Read tweets, search, threads
- Post tweets, replies
- Like/retweet, follow/unfollow
- DMs via browser

### Setup
```bash
openclaw browser start
openclaw browser open https://x.com
# Login manually, then agent can help interact
```
