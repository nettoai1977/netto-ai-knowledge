## üîß How to Integrate (With Antigravity)

Antigravity uses `~/.gemini/antigravity/skills/` as the global skills path. Compatible tools:

1. **Download skills** - Clone repository:
   ```bash
   git clone https://github.com/VoltAgent/awesome-agent-skills.git
   ```

2. **Copy skills to Antigravity** - Example:
   ```bash
   # Copy Anthropic MCP builder
   cp -r awesome-agent-skills/skills/anthropics/mcp-builder ~/.gemini/antigravity/skills/
   
   # Copy Anthropic webapp testing
   cp -r awesome-agent-skills/skills/anthropics/webapp-testing ~/.gemini/antigravity/skills/
   
   # Copy Anthropic skill creator
   cp -r awesome-agent-skills/skills/anthropics/skill-creator ~/.gemini/antigravity/skills/
   ```

3. **Verify skills** - Skills directory:
   ```bash
   ls -l ~/.gemini/antigravity/skills/
   # Should show mcp-builder, webapp-testing, skill-creator
   ```
